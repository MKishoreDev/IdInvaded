# InvadedBot

A Telegram automation bot inspired by the **Sibyl System** from *Psycho-Pass*.

Built using **Python** and **Pyrogram**.

![Status](https://img.shields.io/badge/status-archived-red)
![Python](https://img.shields.io/badge/python-3.x-blue)
![Framework](https://img.shields.io/badge/framework-pyrogram-yellow)

</div>

---

## Overview

InvadedBot was originally created in **2022** by:

* MKishoreDev
* Ryu120
* AuraMoon55 (Anshul)

The project aimed to bring the concept of the **Sibyl System** into Telegram through moderation, automation, administration, and utility features.

During its development, the team built multiple Telegram bots, userbots, and automation systems, with InvadedBot becoming one of the largest collaborative projects among them.

---

## Features

* Modular plugin system
* Group administration tools
* Moderation utilities
* Automation commands
* User management
* Utility functions
* Pyrogram-based architecture
* Extensible codebase

---

## Tech Stack

* Python
* Pyrogram
* MongoDB
* Telegram MTProto API
